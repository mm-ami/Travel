# [Travel](https://mm-ami.github.io/Travel/)
海外パッケージツアーの架空サイトを作成いたしました。<br>
各地の写真の鮮やかさを強調したかったため、色は白と黒をメインに作成いたしました。<br>
ScrollObserverを使用し、画像に動きをつけ、画像から文章に目が移るようにしています。
